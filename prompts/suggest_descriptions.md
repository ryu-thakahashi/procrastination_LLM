## タスク説明候補の生成

与えられたタスク名と先延ばし原因をもとに、ユーザーの先延ばし状況を具体的に説明する文案を5つ提案してください。

## 制約

- 各説明文は 1〜2 文で、自然な日本語とすること。
- 「〜を先延ばしにしている」「〜に先延ばしできない」など、実際の状況を表す形式とすること。
- 5 つの説明文は多様な視点から提案し、似たような内容を重複させないこと。
- JSON 形式のみで出力し、説明文やコードフェンスは含めないこと。

## 出力形式

{{
    "descriptions": [
      "説明文1",
      "説明文2",
      "説明文3",
      "説明文4",
      "説明文5"
    ]
}}

## 例1

### 入力

{{
    "title": "論文執筆",
    "selected_cause": "タスクの解像度: 論文を書き終えるまでに必要なタスクを正確に把握できていない"
}}

### 出力

{{
    "descriptions": [
      "Social Networks の論文執筆を先延ばしにしている。全体の構成や各セクションで必要な分析が多すぎて、どこから始めればよいかわからない状態が続いている。",
      "論文を書く必要があるのは理解しているが、まず何をリサーチすべきかが不明確で、作業を開始する一歩が踏み出せていない。",
      "執筆の全体像が見えていないため、現在の進捗状況がどの段階なのか把握できず、完了までにあとどのくらい必要かわからない。",
      "論文の骨組みが定まっていないため、細部を詰める作業に入ることができず、完成までの道筋が見えていない。",
      "執筆タスクが複雑に感じられて圧倒されており、作業を細分化して段階的に進める計画が立てられていない。"
    ]
}}

## 例2

### 入力

{{
    "title": "就活のES作成",
    "selected_cause": "感情的なハードル: 完成させたのに不合格になったときのことを考えると怖い"
}}

### 出力

{{
    "descriptions": [
      "サイバーエージェントへ提出する ES 作成を先延ばしにしている。せっかく書いたのに不合格になってしまったら、その落胆が大きいと思うと、書く勇気が出ない。",
      "心血を注いで ES を完成させても、評価されなかったら辛いという思いが頭をよぎり、作成に踏み切れていない。",
      "ES を提出して落選する可能性を想像すると不安が大きく、その失敗のリスクを考えると行動に移せない。",
      "完成までこぎつけたとしても、採用試験で失敗するかもしれないというシナリオが浮かんで、書く決心がつかない。",
      "努力が報われない可能性に対する恐怖心があり、そうであれば最初から挑戦しない方が傷つかないと考えてしまっている。"
    ]
}}
