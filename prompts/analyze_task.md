## タスク分析の実施

与えられたタスクの情報と先延ばしの原因をもとに、タスク分析を実施してください。

## 分析方針

- selected_cause に基づき、タスクが先延ばしされている根本原因を特定する
- 障壁（obstacles）を洗い出す
- 原因の種類に応じて、適切な解決策のタイプ（solution_type）を判断する

## solution_type の定義

以下のいずれかを出力してください：

- `action_plan`: タスクの解像度が低い、タスク粒度が粗い → 詳細なアクションプランで解決
- `emotional_reframing`: 感情的なハードルが高い → フレーミング変更で心理的障壁を低減
- `reward_optimization`: 報酬優先性が高い → 短期報酬を設計して動機付け
- `interpersonal`: 対人要因がある → コミュニケーション・フィードバック活用
- `condition_recovery`: コンディション悪化（疲れ、体調）→ 回復策と段階的再開

## 例1

### 入力

{{
    "title": "論文執筆",
    "description": "論文を書き終えるまでに必要なステップやリサーチ範囲が明確になっていないため、次の具体的なアクションが定まらず、手が止まってしまっている。",
    "selected_cause": "タスクの解像度: 次の具体的なアクションが定まらず、手が止まってしまっている"
}}

### 出力

{{
    "title": "論文執筆",
    "description": "論文を書き終えるまでに必要なステップやリサーチ範囲が明確になっていないため、次の具体的なアクションが定まらず、手が止まってしまっている。",
    "selected_cause": "タスクの解像度: 次の具体的なアクションが定まらず、手が止まってしまっている",
    "minimum_goal": "論文の構成案（Introduction, Methods, Results, Discussion の各章に何を書くかのメモ）を完成させる",
    "cause_summary": "タスクが大きすぎて、次にやるべき具体的なアクションが見えないため、心理的に着手しづらくなっている",
    "obstacles": [
        "どこから手をつければいいかわからない",
        "全体のリサーチ範囲やスコープが決まっていないため、必要な作業量が不明",
        "論文全体のステップが明確でないため、一つのタスクが大きすぎて心理的コストが高い"
    ],
    "solution_type": "action_plan"
}}
