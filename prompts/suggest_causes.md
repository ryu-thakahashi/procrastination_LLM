## 先延ばし原因の提案

与えられたタスクに対して、ユーザーが先延ばしにしてしまう原因として考えられるものを5つ提案してください。

## 制約

- 各原因は **カテゴリ名（5つ）** と **具体的な説明文（20〜50文字）** のペアで記述すること。
- 原因は多様な視点から提案し、似たような内容を重複させないこと。
- JSON形式のみで出力し、説明文やコードフェンスは含めないこと。

## 出力形式

{{
    "causes": {{
      "タスクの解像度": "説明文",
      "感情的なハードル": "説明文",
      "報酬優先": "説明文、手を付けようと思わない",
      "対人要因": "説明文",
      "コンディション": "説明文",
    }}
}}

## 例1

### 入力

{{
    "title": "論文執筆",
    "description": "Social Networks に出す論文の執筆を先延ばしにしてしまう。",
}}

### 出力

{{
    "causes": {{
      "タスクの解像度": "論文を書き終えるまでに必要なタスクを正確に把握できていない",
      "感情的なハードル": "完成した時に受けるフィードバックが怖い",
      "報酬優先": "他にもっと楽しいタスクがあるので、手を付けようと思わない",
      "対人要因": "指導教員や共著者等との連絡が面倒",
      "コンディション": "疲れ等によって、体のコンディションが整っていない",
    }}
}}

## 例2

### 入力

{{
    "title": "就活のES作成",
    "description": "サイバーエージェントに提出するESの作成を先延ばししてしまう。"

{{

      "報酬優先": "他にもっと楽しいタスクがあるので、手を付けようと思わない",
      "対人要因": "先輩やメンター等との連絡が面倒",
      "コンディション": "疲れ等によって、体のコンディションが整っていない"
    }}
}}
